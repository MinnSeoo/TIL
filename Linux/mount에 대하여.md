# **mount에 대하여**

리눅스에서는 하드디스크의 파티션, CD/DVD, USB 메모리 등을 사용하려면 특정한 위치에 
연결을 해 줘야 한다. 이렇게 **물리적인 장치를 특정한 위치에 연결시켜주는 과정**을 마운트라고 한다.

쉽게 말하면 디스크 공간과 디렉토리를 연결하는 것 이다.

<br>

**[기본형태]**

`mount -t [파일 시스템] [디바이스이명] [디렉토리]`
